# PyDigiKey
This project visualizes the data from DigiKey in Qt-based UI

### Hardware
NPX UWB (Ultra Wideband) Remote Control  
CAN-USB interface

### Software
Python 3  
Qt QML (via PySide2 (Qt for Python))

create a virtual environment if needed, 
```
pip install PySide2
```
